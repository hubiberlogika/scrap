Frontend React untuk Scrapper Home

Tech stack:
- React 18
- Tailwind CSS 3 (mobile-first responsive)
- Parcel 2 (bundler)
- Axios (HTTP client)

Quick start:

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start dev server (runs on http://localhost:3000):

```bash
npm start
```

Notes:
- Ensure backend is running on http://localhost:5000
- Frontend uses mobile-first design with Tailwind CSS breakpoints (sm:, md:, lg:)
- Responsive components: PropertyCard (expandable on mobile), EditModal (slide-up on mobile)
- Features: property listing with filters, edit modal for updating missing data, error handling, loading states
